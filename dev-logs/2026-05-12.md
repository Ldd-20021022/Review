## 遇到的问题
- Python 3.8 不支持 `str | None` 和 `list[X]` 语法，已改用 `from __future__ import annotations` + `typing.List` / `typing.Optional`
- bcrypt 5.0 在 Windows 上 DLL 加载失败，降级到 4.0.1 解决
- Windows Application Control 策略阻止 Vite/rollup 原生 .node 二进制加载，切换为 CDN + ES Module importmap 方案
- 旧 uvicorn reload 进程缓存导致新路由未加载，需 taskkill 后重启
